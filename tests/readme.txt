Чтобы запустить эти тесты требуется сделать .sh файл 
исполняемым.
На линукс системах это делается с помощью команды 
chmod +x test.sh

В нем надо указать имя вашей программы prog="" (обычно ./a.out),
а также местпоположение папки с тестами, по умолчанию уже установлено

a.txt   идеальная матрица
a20.txt та же матрица, но умноженная на константу
b.txt   вырожденная матрица
c.txt d.txt e.txt невырожденные матрицы с вырожденными блоками
f.txt   меньше данных, чем требуется
